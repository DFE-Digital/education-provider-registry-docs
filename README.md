# Education Provider Registry Docs

This repository contains source-controlled documentation, behaviour definitions, business-domain material and data-model artefacts for the Education Provider Registry.

Published docs site:

[https://dfe-digital.github.io/education-provider-registry-docs/](https://dfe-digital.github.io/education-provider-registry-docs/)

Source locations:

- `features/` - agreed behaviour in Gherkin.
- `business-domain/` - business-focused domain documentation.
- `models/` - vocabulary, taxonomy, ontology and SHACL model artefacts.
- `docs-site/` - Nanoc site used to publish readable generated views.

The source files remain the source of truth. The published site is a readable view generated from those files.
