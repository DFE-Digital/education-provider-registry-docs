# Details Tab Fields Public Sample Validation

## Purpose

This document validates details-tab-fields-csharp-resolution.md by fetching a sample of public GIAS Details pages for each establishment type and scraping the rendered Details-tab field labels.

## Methodology

- Sampled up to 5 random establishments per establishment type from the local extract data.
- Used a deterministic random seed so the sample can be reproduced.
- Fetched https://www.get-information-schools.service.gov.uk/Establishments/Establishment/Details/{urn}#school-dashboard for each sampled URN.
- Scraped dt.govuk-summary-list__key labels from the Details tab HTML.
- Unioned the scraped labels across the sampled URNs for each establishment type.

## Important Limitations

- This is still sampled evidence, not a complete crawl of every establishment.
- The public page may suppress fields that are present in extract data or legacy field/type metadata.
- Some fields are relationship, service-link or feature-flag dependent and may not appear in a small sample.
- Rows with fewer than 5 fetched URNs either had fewer available extract rows or fetch errors.
