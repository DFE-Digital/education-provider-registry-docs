# Education Provider Vocabulary — Real-World References

Each vocabulary concept is classified as one of:

| Category | Meaning |
|---|---|
| **Real-world** | Named and defined on GOV.UK. The concept exists outside GIAS. |
| **Real-world (limited)** | The underlying concept is real, but this specific label or subdivision is not used or defined on GOV.UK. |
| **GIAS construct** | Internal to GIAS — an administrative label, catch-all, data management grouping, or system identifier with no GOV.UK equivalent. |

## 8. Vocabulary-Level Constructs

These concepts belong entirely to the GIAS vocabulary model. No GOV.UK counterpart exists at this level of abstraction.

| Term | Category | Notes |
|---|---|---|
| EstablishmentTypeGroup | GIAS construct | GIAS taxonomy layer grouping establishment types into eleven named sectors. GOV.UK does not publish a parallel classification scheme. |
| RecordCurrencyAndStewardship | GIAS construct | GIAS data stewardship concept. No GOV.UK definition. |
| DateLastChangedOrConfirmed | GIAS construct | GIAS record-management field. No GOV.UK definition. |
| EstablishmentGroupMembership | GIAS construct | GIAS model concept for the relationship between establishments and groups. |
| GroupRelationship / GroupRelationshipType | GIAS construct | GIAS model concept for group-to-group history (e.g. SAT → MAT). The underlying events are real; the relational model is GIAS. |
| Successor MAT / Predecessor SAT | GIAS construct | GIAS group relationship type values. Real events, GIAS labels. |
