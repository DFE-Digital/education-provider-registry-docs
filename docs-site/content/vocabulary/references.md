# Education Provider Vocabulary — Real-World References

Each vocabulary concept is classified as one of:

| Category | Meaning |
|---|---|
| <a id="real-world"></a>**Real-world** | Named and defined on GOV.UK. The concept exists outside GIAS. |
| <a id="real-world-limited"></a>**Real-world (limited)** | The underlying concept is real, but this specific label or subdivision is not used or defined on GOV.UK. |
| <a id="gias-construct"></a>**GIAS construct** | Internal to GIAS — an administrative label, catch-all, data management grouping, or system identifier with no GOV.UK equivalent. |
